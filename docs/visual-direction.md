# 视觉方向

## 品牌形象

项目主视觉从“小鹿”调整为“小狗”。

角色参考图：

```text
D:\健身小项目\assets\references\puppy-ui-reference.png
```

后续所有小狗素材以这张图为准：奶油白毛色、浅粉色耳朵、圆脸、小黑豆眼、淡粉腮红、柔和微笑、趴在卡片边缘的陪伴感。

姿势参考：

```text
D:\健身小项目\assets\references\puppy-cheer-reference.png
D:\健身小项目\assets\references\puppy-cross-reference.png
```

- `puppy-cheer-reference.png`：用于“最低线”卡片，开心鼓励、双手举起、旁边有星星。
- `puppy-cross-reference.png`：用于“不要做”卡片，双手交叉，温和制止。

原因：

- 小狗更贴近日常陪伴感，适合“什么都不懂也有人带着做”的产品定位。
- 小狗表情更容易扩展出鼓励、提醒、拒绝、疲惫、开心等状态。
- 小狗更适合远程陪伴式产品，不会显得过度童话或装饰化。

## 需要的核心素材

- 小狗头像 logo
- 点赞小狗
- 鼓励小狗
- 双手交叉/制止小狗
- 疲惫安慰小狗
- 训练女生插画
- 健康餐盘插画
- 白班/夜班/休息/很累状态图标
- 健身房/家里/便利店地点图标
- 底部导航图标

## 风格要求

- 可爱但不低幼
- 像真实移动端 App，不像表情包合集
- 小狗统一角色：同一毛色、同一脸型、同一比例，必须接近参考图
- 主色仍用粉色，辅色用薄荷绿、柠檬黄、淡紫
- 素材优先透明 PNG/WebP，图标优先 SVG
