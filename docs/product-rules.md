# 女生生活化减脂项目规则库

## 夜班后睡前补给

适用场景：用户值完夜班，准备回家睡觉前，出现疲劳、饥饿、发虚、低血糖感，或者距离上一餐已经很久。

核心原则：吃一点轻、好消化、带蛋白质和少量碳水的东西，帮助缓解疲劳和恢复；不要吃油腻、辛辣、过量正餐，避免增加胃负担和影响睡眠。

推荐组合：

| 组合 | 适合情况 |
| --- | --- |
| 温牛奶或无糖酸奶 + 香蕉 | 最省事，适合疲劳、饥饿、轻微发虚 |
| 牛奶 + 茶叶蛋或水煮蛋 1-2 个 | 想补蛋白，但不想吃正餐 |
| 无糖酸奶 + 少量燕麦 | 夜班后很饿，但马上要睡觉 |
| 豆浆 + 鸡蛋 | 不想喝奶或不方便买牛奶 |
| 少量鸡胸肉或卤牛肉 + 半个玉米或红薯 | 夜班消耗大、明显发虚 |
| 小米粥或燕麦粥 + 鸡蛋 | 胃不舒服、想吃热的 |

默认最小执行方案：

```text
一盒奶 + 一个香蕉 + 一个鸡蛋
```

或：

```text
无糖酸奶 + 少量燕麦 + 一个鸡蛋
```

禁止或不推荐：

- 炸鸡、鸡排、烧烤
- 泡面、辣条、重辣外卖
- 奶茶、甜饮料
- 很大一顿正餐
- 大量坚果
- 空腹咖啡

生成计划时的规则：

- 如果用户选择 `夜班后`，默认不安排正式训练。
- 如果用户选择 `夜班后 + 马上睡觉`，只生成睡前补给，不生成大餐。
- 如果用户反馈 `头晕 / 手抖 / 心慌`，优先推荐 `香蕉 + 牛奶` 或 `玉米 + 鸡蛋`。
- 如果用户频繁出现夜班后低血糖感，提示班中饮食也需要调整，不能只靠下班后补。
