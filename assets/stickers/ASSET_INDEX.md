## 风格参考与备用素材

### cute-energy

路径：

```text
D:\健身小项目\assets\stickers\cute-energy\illustrations_clean
```

来源：

```text
D:\健身小项目\assets\source-boards\cute-energy-korean-board.jpg
```

用途：

- UI 插画风格参考
- 情绪陪伴素材
- 状态提示：加油、晚安、治愈、很累、工作中、到达、OK
- 如果 cat-companion 不够用，可从这里补同类贴纸氛围素材

## 使用规则

- 优先从 `cat-companion` 调用主视觉插画。
- `cute-energy` 作为陪伴氛围和状态提示备用。
- 不再固定小狗、小鹿或单一 IP。
- UI 风格以手账贴纸、浅色卡片、温柔陪伴感为主。
- 插画只做辅助，不要让素材压过“今日计划生成”的主操作。
